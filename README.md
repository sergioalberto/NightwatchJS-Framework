# NightwatchJS-Framework
This a project that contains the basic setup to start an automation project with NightwatchJS.

## Docker Usage
```
make build
make run browser=Chrome
make debug browser=Firefox
make clean
```
or
```
runDocker.bat
stopDocker.bat
```
